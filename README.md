# vproject

A template for making projects with nodejs, webpack, babel, eslint and jsbeautify.

## Installation

```bash
npm install vproject
```

## Node usage

```js
const { vproject } = require('vproject');
```

## Browser usage

```html
<script src="path/to/vproject.js"></script>
```

```js
const vproject = window.vproject;
```
